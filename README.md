# ExpressGo Logistics

ExpressGo is a motorcycle courier and same-day delivery service operating across Istanbul.

## Service Areas

- Esenyurt
- İkitelli
- Bahçeşehir
- Beylikdüzü
- Güneşli

## Services

- Same-day delivery
- Urgent document courier
- Parcel delivery
- Corporate logistics
